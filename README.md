# 📈 全球股市 AI 投資助手：整合 yfinance 與動態視覺化分析

## 📌 專案簡介

本專案旨在為投資者提供一個**跨市場（台股與美股）**的即時監測與深度分析工具。透過串接 Yahoo Finance API 獲取海量金融數據，結合技術分析指標（MA）、動態 K 線與配息數據追蹤，協助用戶在複雜的市場環境中，快速識別強勢標的並評估長期投資價值。

---

## 🛠️ 技術堆疊

* **開發環境**: Python 3.11
* **前端介面**: **Streamlit** (即時響應式網頁框架)
* **數據獲取**: **yfinance** (Yahoo Finance API 封裝)
* **數據處理**: Pandas, NumPy
* **動態視覺化**:
* **Plotly**: 建立高互動性 K 線圖（Candlestick）、成交量與均線圖，支援自定義 Hover 提示。


* **自動化集成**: GitHub Actions (用於 LINE Bot 每日推播自動化)

---

## 📊 核心功能與實作流程

### 1. 即時市場看板 (Real-time Market Dashboard)

* **自動篩選**: 針對台/美股權重股與熱門 ETF 進行即時漲跌幅排序。
* **交互跳轉**: 點擊排行榜即可自動切換至該標的的「深度分析」模式。

### 2. 深度技術分析 (Technical Analysis Insight)

* **多週期 K 線**: 支援日 (1d)、週 (1wk)、月 (1mo) 不同粒度的行情觀察。
* **指標回測**: 提供可調式短期與長期移動平均線 (MA)，視覺化趨勢交會點。

### 3. 特徵提取與預處理

* **日期格式化**: 針對週/月線數據，自動計算「週起始日」與「月顯示格式」，優化用戶懸停檢查體驗。
* **數據補全**: 處理 ETF 配息數據中的缺失值，並根據市場特性（台/美）自動推算預計發放日。

### 4. 股利政策與殖利率追蹤

* **動態頻率判斷**: 自動辨識月配、季配或半年配標的。
* **收益估算**: 基於最新收盤價實施年化殖利率試算，為存股族提供決策參考。

## 📈 視覺化成果

### 1. 動態互動 K 線圖

整合了價格行為與成交量，並透過 **Plotly** 實現 `x unified` 游標鎖定，讓用戶在觀察價格時能同步對準下方的成交量變化。

### 2. 多維度配息紀錄表

將繁瑣的歷史配息轉化為結構化表格，包含除息日、配息金額、推估發放日及預估年化殖利率。

## 💡 投資策略與洞察 (Investment Insights)

| 分析模組 | 監測指標 | 應用場景 |
| --- | --- | --- |
| **趨勢追蹤** | MA 短長線交叉 | 判斷黃金/死亡交叉，設定買賣基準點。 |
| **資金流向** | 成交量與價格配合 | 識別帶量突破或無量盤整，避開假突破陷阱。 |
| **收益分析** | 預估年化殖利率 | 評估高股息 ETF (如 00878, 00919) 的進場時機與穩定性。 |
| **市場輪動** | 漲跌排行榜 | 觀察當日領漲板塊（如 AI 概念股、金融股）的強弱變動。 |
